# PowerShell-Scripts
Production-ready PowerShell scripts for Microsoft 365, Intune, Entra ID, and endpoint management

## Scripts

### Get-DeviceReadiness.ps1
Checks whether a device meets Windows 11 and Autopilot requirements:
- Windows edition
- Windows version and build
- TPM status
- Secure Boot

### Connect-GraphAutopilot.ps1
Connects to Microsoft Graph for Autopilot management.
